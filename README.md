Рукописный трансформер-переводчик на основе архитектуры Encoder - Decoder (All you need is attention-like)

- model.py — encoder-decoder Transformer с MHA, FFN, positional encoding, encoder/decoder blocks.
- masks.py — causal/padding masks.
- inference.py — greedy autoregressive translation.
- tokenizer.py — BPE tokenizer (из Hugging Face).
